Amazing Favorites 是一款浏览器扩展，它可以帮助你高效的管理你的浏览器收藏，并且提供对应的搜索功能。让您可以在最短的时间内找到您想要的信息。

它包含有一些基本的功能：

添加收藏

![添加收藏](/images/20210605-001.png)

快速搜索

![快速搜索](/images/20210605-002.png)

另外，您还可以在控制面板中开启一些基于云端的高级功能：

1. 支持拼音搜索
2. 支持云端同步收藏夹标签

![控制面板](/images/20210605-003.png)
